# Security Enrichment: CVE-2024-5678

## Executive Summary

CVE-2024-5678 is a **Medium** severity Information Disclosure vulnerability affecting Example Web Framework 1.0.0 through 1.2.5. The vulnerability allows authenticated users to access sensitive configuration files through directory traversal.

**CVSS Base Score:** 5.3 (Medium)
